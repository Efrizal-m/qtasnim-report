# qtasnim-product-management

Backend: Nest.js
to run locally: npm run start:dev

Frontend: react.js
to run locally: npm run start


add this to .env file in /qtasnim-be
PORT=3001
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASS=postgres
DB_DIALECT=postgres
DB_NAME_TEST=qtasnim
DB_NAME_DEVELOPMENT=qtasnim
DB_NAME_PRODUCTION=qtasnim
